Senior Design Challenge Branding

I worked with Professor Eugene Korsunskiy and Ashley Manning on branding Thayer Engineering's new class ENGS 16: Senior Design Challenge, which is a capstone class for the design thinking experience. I created the logo and promotional posters for the class. For the logo, I wanted to incorporate various aspects of design thinking (such as the iconic sticky notes, hexagon shape, or intersection Venn Diagram circles). We wanted a minimalistic logo that would transfer well in a single solid color. In the end, we created the Senior Design Challenge logo with intersection Venn Diagram circles that include a hexagon element in the center. Afterwards, I created some posters to promote class enrollment. A few drafts are pictured below, but we eventually went with the design that features a top-down view of a table with various tools scattered around.
